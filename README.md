# Stress Detection with Machine Learning

This project detects whether a social media post indicates psychological stress.

## Problem Type
- Binary Classification
- Labels:
  - 0 → No Stress
  - 1 → Stress

## Dataset
- Source: Reddit mental health subreddits
- Used columns:
  - `text`
  - `label`

## Model
- Support Vector Machine (SVM)
- NLP pipeline (vectorization + classifier)
- Best accuracy: **70%**
